# Random Name Picker

A fun, interactive random name picker with multiple animation styles. Hosted entirely on GitHub Pages.

## Features

- **2 Animation Styles:**
  - **Spin Wheel** - Classic spinning wheel of fortune with colorful segments
  - **Slot Machine** - Vegas-style slot machine animation

- **Easy Name Management:**
  - Add names one at a time
  - Bulk add multiple names (one per line)
  - Remove individual names
  - Clear all names

- **Winner Popup:**
  - Popup announces the winner
  - Remove winner from list
  - Confetti celebration effect

- **Persistent Storage:**
  - Names are saved in browser's localStorage

- **Responsive Design:**
  - Works on desktop, tablet, and mobile

## Quick Start

1. Clone or download this repository
2. Open `index.html` in your browser
3. Add names in the sidebar
4. Choose your animation style (Wheel or Slots)
5. Click "Pick Name!"
